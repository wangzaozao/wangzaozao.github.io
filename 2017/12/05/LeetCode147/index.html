LeetCode147. Insertion Sort List

发表于 2017-12-05

Description

Sort a linked list using insertion sort.

Demo1(链表的插入排序)

思路：
本题是插入排序的链表版本。

传统数组版本做法就是两重循环，第一重是遍历所有元素，第二重是遍历已排序部分进行插入。

链表版本类似，在遍历每个元素过程中，遍历已排序部分进行插入。
